Computer System

A computer system is a basic, complete and functional hardware and software setup with everything needed to implement computing performance. A computer system consists of a motherboard, processor, Random accessing memory (RAM) and a storage. Computer system can be found in many forms such as laptops, desktop, mainframes and even in phones. Smart phones are considered as a computer due to its ability to enable us to communicate with the Internet. As time flies, the processor of a computer system is getting smaller, thus, smaller things can be called as a computer system!
